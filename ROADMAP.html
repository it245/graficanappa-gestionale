v1.0 MES Completo — Fondamenta, Produzione, Costi Gen-Feb 2026

Sistema gestionale MES completo: dalla gestione ordini all'analisi dei costi, passando per produzione, scheduling, integrazione Onda/Prinect e reportistica direzionale.

Architettura & Modello Dati

  • Architettura Laravel 11 con MySQL — deploy su Windows Server / Apache
  • Modello dati: Ordini, OrdineFasi, Operatori, Reparti, FasiCatalogo, CostoReparto, PrinectAttivita, pivot tables
  • Login operatori con codice operatore + login admin protetto da password hashata
  • Middleware AdminAuth — protezione route /admin/*
  • Pagina errori personalizzate (403, 404) e health check endpoint (/health)

Produzione & Operatori

  • Dashboard Owner — vista completa di tutte le fasi attive, ordinate per priorità
  • Dashboard Operatore — fasi filtrate per reparto assegnato
  • Produzione: avvio, pausa, termine fase con tracciamento operatore e timestamp
  • Cascata stati fasi automatica (FaseStatoService): quando una fase termina, la successiva diventa “pronta”
  • Editing inline campi ordine/fase via AJAX (qta, note, stato, date, carta...)
  • Dettaglio commessa (owner + operatore) con note prestampa, responsabile, commento produzione
  • Vista fasi terminate con storico
  • Formula priorità: giorni_rimasti − (ore/24) + (ordineFase/10000)
  • Catalogo 96+ fasi di produzione con tempi avviamento e copie/ora
  • Mapping fasi → reparti (stampa offset, digitale, plastificazione, fustella, legatoria, piegaincolla, stampa a caldo, spedizione, esterno...)

Integrazione Onda (ERP)

  • Sincronizzazione automatica con Onda IQ (SQL Server) — import ordini + fasi da ERP
  • Sync schedulata ogni ora (lun-sab 06:00–22:00) + bottone manuale
  • Auto-creazione reparti e fasi catalogo da Onda se mancanti
  • Ricalcolo automatico ore/priorità dopo ogni sync
  • Import campi personalizzati: Note Prestampa, Responsabile, Commento Produzione
  • Import prezzi vendita (TotMerce) e costi materiali (SUM PRDDocRighe.Totale) da Onda
  • Import ordini da Excel (.xlsx) con mapping fasi e deduplicazione
  • Sync Excel bidirezionale: nuove righe e eliminazione righe mancanti

Integrazione Prinect (Stampa Offset)

  • Dashboard Prinect live con auto-refresh AJAX (polling 10s)
  • Storico attività filtrabile con sync automatica
  • Report per commessa: breakdown per workstep e operatore
  • Correlazione automatica job Prinect ↔ commessa MES
  • Aggiornamento automatico fogli_buoni / fogli_scarto nelle fasi STAMPAXL106
  • Calcolo OEE (disponibilità, performance, qualità) per macchina stampa
  • Alert automatici: macchina ferma, scarti sopra soglia, OEE critico

Scheduling & Spedizioni

  • Scheduling Produzione — interfaccia Gantt con 3 viste (per macchina, per commessa, tabella priorità)
  • Turni specifici per reparto (offset 0-23, piegaincolla/fustella 6-22, stampa a caldo 6-22 + sab 6-13, altri 8-17)
  • KPI scheduling: commesse attive, in tempo, in ritardo, critiche
  • Dashboard Spedizione dedicata: Da spedire / Spedite oggi / In attesa
  • Forza consegna: chiude forzatamente tutte le fasi e completa la spedizione

Admin & Gestione

  • Dashboard Admin: elenco operatori con codice, ruolo, reparti, stato attivo
  • CRUD operatori da admin: crea, modifica, attiva/disattiva
  • Configurazione tariffe orarie per reparto con storicizzazione (valido_dal / valido_al)

Statistiche, Report & Costi

  • Statistiche operatori: fasi completate, ore lavorate, quantità prodotte con grafici Chart.js
  • Report commessa: riepilogo tempi effettivi vs stimati per ogni fase
  • Lista commesse con filtro completate/in corso + storico archivio
  • Report produzione settimanale stampabile (KPI, top operatori, commesse in ritardo)
  • Report Direzione multi-periodo (settimana/mese/trimestre/semestre/anno): 6 KPI con delta %, colli di bottiglia, trend produzione, performance operatori, analisi pause, scarto Prinect, clienti, commesse in ritardo — 5 grafici + export Excel 5 fogli
  • Report Prinect multi-periodo: produzione offset, scarto %, OEE breakdown, trend, performance operatori, top commesse — 5 grafici + export Excel 3 fogli
  • Report Costi & Margini multi-periodo: calcolo costo effettivo commessa (ore_effettive × tariffa_oraria per fase), costo materiali da Onda, margine vendita, 6 KPI con delta %, top profittevoli / in perdita, costi per reparto — 2 grafici + export Excel 4 fogli
  • Granularità adattiva trend: giorno (settimana/mese), settimana (trimestre/semestre), mese (anno)
  • Export report in PDF (window.print con CSS @media print ottimizzato A4)
  • Dashboard KPI giornaliero owner: fasi completate oggi, ore lavorate, spedizioni

v2.0 Notifiche & Mobilità Apr-Giu 2026

Sistema di notifiche real-time e accesso mobile per operatori e titolare. L'obiettivo è eliminare la carta dal reparto.

  • Progressive Web App (PWA): installabile su tablet e smartphone operatori
  • Scansione barcode/QR per avvio/termine fase da dispositivo mobile
  • Notifiche push (browser) per commesse in ritardo critico
  • Notifica automatica al titolare quando una commessa viene spedita
  • Dashboard sintetica su TV in reparto (kiosk mode, rotazione automatica)
  • Interfaccia responsive completa per tutti i ruoli
  • Notifica Telegram/WhatsApp al titolare (riepilogo giornaliero serale)
  • WebSocket real-time (Laravel Reverb) per aggiornamento istantaneo dashboard

v3.0 Claude AI Intelligence Lug-Set 2026

SUPER INTEGRAZIONE: Intelligenza Artificiale nel MES
Integrazione di Claude (Anthropic) come motore AI. L'AI analizza dati di produzione storici e in tempo reale per fornire insight, previsioni e suggerimenti operativi.
  • Integrazione Claude API (Anthropic SDK) nel backend Laravel
  • Assistente AI per il titolare: chat in linguaggio naturale per interrogare i dati di produzione
    • “Quali commesse sono a rischio questa settimana?”
    • “Confronta la produttività di febbraio con gennaio”
    • “Perché la commessa 0066550 ha sforato i tempi?”
  • Analisi predittiva ritardi: AI analizza pattern storici per prevedere commesse a rischio
  • Ottimizzazione tempi stimati: AI ricalibra avviamento e copie/ora in base ai dati reali accumulati
  • Report AI generati: il report settimanale include osservazioni, anomalie e raccomandazioni scritte dall'AI
  • Anomaly detection: segnalazione automatica di tempi anomali (fase troppo lunga/corta rispetto allo storico)
  • Cache intelligente delle risposte AI (Redis) per contenere i costi API

v4.0 Pianificazione Intelligente Ott-Dic 2026

SUPER INTEGRAZIONE: Scheduling AI-Powered
L'AI pianifica la produzione considerando vincoli reali: disponibilità macchine, skill operatori, urgenze, setup time e storico performance.
  • Scheduling automatico AI: piano di produzione ottimale con minimizzazione ritardi e setup
  • Drag & drop fasi nel Gantt per riordinamento manuale + ricalcolo AI impatto
  • Simulazione “what-if” AI: “Se arriva una commessa urgente da 5000 fogli, cosa succede al planning?”
  • Gestione manutenzione macchine (fermi programmati integrati nel Gantt)
  • Previsione carico lavoro settimanale per reparto con suggerimento straordinari/turni
  • Assegnazione intelligente operatori: AI suggerisce chi assegnare a quale fase in base a skill e velocità storica
  • Ottimizzazione sequenza lavori per minimizzare tempi cambio formato/lastre

v5.0 Qualità & Tracciabilità Gen-Mar 2027

Sistema di controllo qualità integrato nella produzione con tracciabilità completa.

  • Checklist qualità per fase: verifiche obbligatorie prima del termine (colore, registro, grammatura, piega)
  • Foto allegata alla fase: operatore scatta foto del prodotto a fine lavorazione
  • Tracciabilità completa: per ogni foglio stampato, sapere chi/quando/quale macchina
  • Gestione non conformità: apertura ticket, assegnazione responsabile, azione correttiva
  • Dashboard qualità: % scarti per macchina, per operatore, per tipologia lavoro
  • Storico reclami cliente collegato alla commessa
  • Certificazioni FSC: tracciabilità catena di custodia carta certificata

v6.0 Magazzino & Materiali Apr-Giu 2027

Gestione magazzino materie prime e semilavorati con integrazione automatica ai consumi di produzione.

  • Anagrafica materiali: carta, lastre, inchiostri, fustelle, plastificazione con giacenza e soglia riordino
  • Scarico automatico materiali a fine fase: carta consumata calcolata da fogli stampati
  • Movimenti di magazzino: carico da fornitore, scarico produzione, rettifiche inventario
  • Alert scorta minima: notifica automatica quando un materiale scende sotto soglia
  • Costo medio ponderato materiali (FIFO) per calcolo costi commessa più preciso
  • Integrazione con Onda per carico automatico da DDT fornitore
  • Report consumi mensili per tipologia materiale e per commessa

v7.0 IoT & Digital Twin Lug-Set 2027

SUPER INTEGRAZIONE: Fabbrica Connessa
Sensori IoT sulle macchine + digital twin dello stabilimento. Il sistema “vede” la fabbrica in tempo reale e, combinato con l'AI, prevede guasti e ottimizza consumi.
  • Sensori IoT: contatori fogli, temperatura, vibrazione su macchine principali (offset, fustella, plastificatrice)
  • Digital Twin 2D: mappa interattiva dello stabilimento con stato macchine in tempo reale
  • Manutenzione predittiva AI: analisi vibrazioni e pattern di utilizzo per prevedere guasti
  • Monitoraggio consumi energetici per macchina e per commessa
  • Conteggio automatico produzione: sensori contano i fogli → aggiornano qta_prod senza input manuale
  • Protocollo MQTT per comunicazione sensori → server → dashboard
  • Storico dati sensori con grafici trend e correlazione con produttività

v8.0 Business Intelligence & Analytics Avanzata Ott-Dic 2027

Dashboard direzionali avanzate con drill-down, benchmark, previsioni finanziarie e analisi multi-dimensionale.

  • Dashboard direzionale interattiva con drill-down (anno → mese → settimana → giorno → commessa)
  • Benchmark per reparto: confronto produttività tra periodi, tra operatori, tra macchine
  • Previsione fatturato e margine trimestrale basata su portafoglio ordini + storico
  • Analisi ABC clienti: classificazione per fatturato, margine, volumi
  • Analisi costi fissi vs variabili per commessa (ammortamento macchine, costo operatore, materiali)
  • KPI personalizzabili dall'admin: scelta metriche, soglie alert, grafici preferiti
  • Export automatico report periodici via email (schedulato)

v9.0 Portale Clienti & API Pubblica Ott-Dic 2027

Accesso clienti per monitorare lo stato dei propri ordini e API REST per integrazione con sistemi esterni.

  • Portale clienti read-only: login cliente, stato commesse, tracking avanzamento fasi
  • Notifica automatica al cliente: commessa in lavorazione, pronta, spedita
  • Download documenti: DDT, certificati qualità, foto prodotto finito
  • API REST pubblica documentata (OpenAPI/Swagger) per integrazione sistemi terzi
  • Webhook per eventi: nuova commessa, fase completata, commessa spedita
  • Integrazione con sistema contabile per fatturazione automatica
  • Single Sign-On (SSO) per accesso unificato Onda + MES + Portale Clienti

v10.0 Industria 4.0 Completa Gen-Feb 2028

TRAGUARDO FINALE: Fabbrica Intelligente Integrata
La versione finale unifica tutte le componenti in un ecosistema coerente: AI che orchestra produzione, sensori che alimentano i modelli predittivi, clienti connessi in tempo reale, e un digital twin che rispecchia fedelmente ogni aspetto dello stabilimento.
  • App nativa iOS/Android per il titolare con notifiche push e dashboard sintetica
  • AI Coach per operatori: suggerimenti personalizzati per migliorare produttività e ridurre scarti
  • Digital Twin 3D navigabile dello stabilimento con dati real-time sovrapposti
  • Gestione completa manutenzione: programmata, preventiva (AI), correttiva con storico interventi
  • Sostenibilità: calcolo impronta carbonio per commessa, report ESG, ottimizzazione consumi
  • Multi-stabilimento: architettura pronta per gestire più sedi produttive
  • Formazione integrata: tutorial guidati per nuovi operatori basati su AI
  • Audit trail completo: ogni azione nel sistema è tracciata e consultabile per compliance
