Integrazioni di Sistema
Sistema Protocollo Direzione Dettagli
Onda IQ (ERP) SQL Server ← Lettura Importazione commesse, clienti, fasi lavorazione. Sync ogni ora.
Prinect (Heidelberg) REST API ← Lettura Tempi avviamento, esecuzione, fogli stampati. Offset XL 106.
Fiery (EFI) REST API ← Lettura Stato job, fogli totali, accounting storico. Stampa digitale.
BRT (Corriere) SOAP ← Lettura Tracking spedizioni, esiti consegna, gestione multi-spedizione.
NetTime (Presenze) File Share ← Lettura Timbrature dipendenti, importazione automatica ogni 5 minuti.
Excel (Produzione) PhpSpreadsheet ↔ Bidirezionale Import/export dati produzione, sync bidirezionale ogni 2 min.
Google Drive File Sync ← Lettura Sincronizzazione file di produzione e documenti condivisi.
