## Roadmap consolidata 90 giorni

### Mese 1 — Setup + Portfolio
- Verifica contratto Grafica Nappa (patto non concorrenza?)
- Verifica RAL CU 2026 <€35k (per forfettario)
- Registra dominio Cloudflare (.com) + OVH (.it)
- Setup email professionale (Cloudflare Routing + Gmail)
- Costruisci 1 sito demo "Pizzeria del Sole" (Astro template) → deploy Cloudflare Pages
- 3 siti gratis/€300 a pizzerie note Casoria → portfolio + testimonial video

### Mese 2 — Outbound
- 5 visite fisiche/settimana zona target
- 20 Instagram DM/giorno pizzerie business
- 10 cold email/giorno SOLO `info@` con template GDPR
- DevDay Napoli + NAStartUp networking
- Registrazione Addlance/ProntoPro (secondario)

### Mese 3 — Scaling
- Intercettare clienti **CCIAA Voucher PID** (€10-15k fondo perduto)
- Diventare fornitore accreditato MIMIT 2026 (finestra 4 marzo – 23 aprile)
- Apri P.IVA forfettario codice 74.12.01 + aliquota 5% (se requisiti OK)
- Iscrizione Gestione Separata INPS
