# Setup fiscale freelance web Italia 2026

> Per Giovanni: dipendente Grafica Nappa, side-hustle siti vetrina nel weekend.
> Dati richiedono **conferma commercialista** prima di operare.

## 1. Prestazione Occasionale (art. 67 TUIR)

| Voce | Valore |
|---|---|
| Limite annuo lordo | **€ 5.000** totale committenti |
| Ritenuta d'acconto | 20% sul lordo (solo se committente è P.IVA/azienda) |
| Dichiarazione | Quadro RL 730/Redditi PF; esonero se redditi <€ 4.800 |
| Carattere | Saltuario, non abituale |
| Documento | Ricevuta non fiscale + bollo €2 sopra €77,47 |

**Pro**: zero costi fissi, niente contributi <€5k, niente fattura elettronica.
**Contro**: tetto basso (3-4 siti), no deduzioni costi, rischio riqualificazione.

## 2. P.IVA Regime Forfettario 2026

| Voce | Valore |
|---|---|
| Soglia ricavi | **€ 85.000** (oltre €100k uscita immediata) |
| Aliquota standard | **15%** sostitutiva |
| Aliquota agevolata | **5% primi 5 anni** se nuova attività |
| Contributi INPS Gestione Separata | **26,07%** sul reddito imponibile |
| Fattura elettronica | **Obbligatoria dal 1/1/2024** |

### Requisiti aliquota 5%

1. No attività professionale/d'impresa nei 3 anni precedenti
2. Non mera prosecuzione lavoro dipendente
3. Se prosecuzione altrui: ricavi anno prec <€85k

**Caso Giovanni**: Grafica Nappa = tipografia/MES. Web design freelance NON è prosecuzione del ruolo IT/MES. Probabile 5% MA: nessun cliente del side-hustle deve essere Grafica Nappa o portato via.

## 4. Compatibilità lavoro dipendente

### Soglia RAL
**€ 35.000 lordi annui** anno precedente (Legge Bilancio 2025, confermata 2026).
Sopra = causa esclusione forfettario.
Verifica CU 2026 (redditi 2025).

### Comunicazione datore
- **Privato** (Grafica Nappa): **nessun obbligo legale**
- Consigliato: comunicazione informale per trasparenza

### Art. 2105 c.c. — Obbligo fedeltà
- Vendere **siti web a terzi** → no concorrenza tipografica → OK
- Vendere stampati/packaging/etichette → concorrenza diretta → vietato
- Clienti di Grafica Nappa portati via → sviamento clientela → vietato

### Patto di non concorrenza (art. 2125 c.c.)
**Rileggi contratto Grafica Nappa**. Per essere valido: scritto, corrispettivo, durata max 5 anni (quadri) / 3 anni (altri), oggetto e zona limitati.

## 6. Contratti + GDPR

### Contratto prestazione opera (artt. 2222-2238 c.c.)

Clausole minime:
1. Oggetto preciso (tipologia, n° pagine, CMS, deliverable)
2. Compenso + acconto **30-50%** alla firma
3. Tempistiche + penali
4. Proprietà intellettuale (a saldo)
5. Hosting/dominio responsabilità
6. Manutenzione post-consegna
7. Risoluzione
8. Nomina Responsabile trattamento art. 28 GDPR (se accesso a dati)
9. Limitazione responsabilità + foro

### GDPR per il cliente
- Informativa privacy art. 13
- Banner cookie + cookie policy
- Registro trattamenti se >250 dipendenti
- Form contatti: checkbox consenso

### Documenti per freelance
- Informativa privacy propria
- DPA con cloud usati (Mailchimp, GW)

**Consiglio**: primi 2-3 contratti con template avvocato digitale (€200-500). Niente fac-simile generico.
